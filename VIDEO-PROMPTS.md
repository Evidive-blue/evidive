# Prompts Vidéo Background - Evidive

Ce document contient les prompts détaillés pour créer des vidéos de fond pour chaque section du site Evidive. Les vidéos doivent être en mode loop (boucle) et s'enchaîner de manière fluide.

## Spécifications Techniques

- **Format** : MP4 (H.264)
- **Résolution** : 1920x1080 (Full HD) minimum, idéalement 4K (3840x2160)
- **Durée** : 15-30 secondes par vidéo (boucle infinie)
- **FPS** : 30 ou 60 fps
- **Style** : Cinématique, sous-marin, ambiance apaisante
- **Transition** : Chaque vidéo doit pouvoir s'enchaîner naturellement avec la suivante
- **Audio** : Optionnel (sans audio recommandé pour éviter les conflits)

---

## 1. HOME PAGE (/)

**Prompt :**
```
Vue sous-marine depuis la surface, caméra descendant lentement dans l'eau turquoise cristalline. 
Rayons de soleil perçant l'eau, créant des reflets dorés et des caustiques sur le fond. 
Bulles remontant doucement. Poissons tropicaux colorés (poissons-clowns, poissons-anges) nageant paisiblement. 
Profondeur : 0-5 mètres. Ambiance lumineuse et accueillante. 
Mouvement de caméra : descente très lente et fluide, comme si on plongeait pour la première fois.
```

**Caractéristiques** :
- Couleurs : Turquoise clair, bleu ciel, reflets dorés
- Profondeur : Surface (0-5m)
- Vie marine : Poissons tropicaux colorés, bulles
- Ambiance : Lumineux, accueillant, découverte

## Guide de Transition entre Vidéos

Pour créer une expérience fluide, chaque vidéo doit se terminer et commencer de manière à pouvoir s'enchaîner :

1. **Home → About** : Transition de surface vers récif (descente continue)
2. **About → Explorer** : Transition récif vers kelp (profondeur croissante)
3. **Explorer → Centers** : Transition kelp vers site structuré (retour vers zone éclairée)
4. **Centers → Contact** : Transition site actif vers baie calme (montée vers surface)
5. **Contact → Login** : Transition baie vers contre-plongée (changement d'angle)
6. **Login → Register** : Transition jour vers nuit (changement temporel)
7. **Register → Dashboard** : Transition nuit vers structure organisée (retour lumière)
8. **Dashboard → Profile** : Transition structure vers portrait (zoom avant)
9. **Profile → Careers** : Transition portrait vers équipe (zoom arrière)
10. **Careers → Onboard** : Transition professionnel vers formation (retour début)

---

## Notes de Production

- **Cohérence visuelle** : Maintenir la même qualité d'image et style cinématique
- **Vitesse de mouvement** : Toujours lent et fluide (pas de mouvements brusques)
- **Stabilité** : Caméra stable, pas de tremblements
- **Loop parfaite** : Chaque vidéo doit boucler sans coupure visible
- **Optimisation** : Compresser pour web tout en gardant la qualité visuelle
- **Format de sortie** : MP4 H.264, 30fps, qualité élevée

---

## Utilisation Technique

Les vidéos seront utilisées comme backgrounds avec :
- Overlay sombre léger pour la lisibilité du texte
- Opacité ajustable selon la section
- Animation de transition entre les vidéos lors du changement de page
- Mode loop automatique
