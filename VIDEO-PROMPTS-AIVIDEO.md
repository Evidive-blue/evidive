# Prompts Vidéo Background - Evidive (Format AIVideo.com)

Ce document contient les prompts optimisés pour la plateforme **aivideo.com/create** avec les paramètres recommandés pour chaque section.

---

## Configuration Générale Recommandée

- **Agent Strategy** : "Consistent Style & Characters" (pour cohérence entre toutes les vidéos)
- **Orientation** : "Landscape (16:9)" (format paysage pour backgrounds)
- **Duration** : 20-30 secondes (boucle parfaite)
- **Voiceover** : Désactivé (vidéos de fond sans narration)
- **Captions** : Désactivé (pas de texte sur les vidéos de fond)
- **Music** : Optionnel (selon préférence, mais généralement sans musique pour éviter les conflits)
- **Sound Effects** : Désactivé (ambiance silencieuse)
- **Media Type** : "AI Video" sélectionné

## Guide d'Utilisation sur AIVideo.com

### Étapes pour chaque vidéo :

1. **Ouvrir aivideo.com/create**
2. **Configurer les paramètres généraux** :
   - Agent Strategy : "Consistent Style & Characters"
   - Orientation : "Landscape (16:9)"
   - Duration : 20-30 secondes
   - Voiceover : **OFF**
   - Captions : **OFF**
   - Music : Laisser vide (ou optionnel)
   - Sound Effects : **OFF**

3. **Sélectionner Media Type** :
   - Cocher "AI Video"

4. **Remplir le Video Script** :
   - Copier-coller le prompt correspondant à la page

5. **Définir le Visual Style** :
   - Soit utiliser un Style Preset existant
   - Soit télécharger une référence de style
   - Soit remplir "Describe your own custom style..." avec la description fournie

6. **Remplir Target Audience** :
   - Utiliser la description fournie pour chaque page

7. **Générer la vidéo** :
   - Cliquer sur "Generate"
   - Vérifier le coût en crédits
   - Télécharger la vidéo générée

8. **Optimiser pour boucle** :
   - Vérifier que la vidéo boucle parfaitement
   - Si nécessaire, ajuster le prompt pour mentionner "seamless loop" plus explicitement

---

## Conseils pour une Cohérence Visuelle

1. **Créer un Style Template** :
   - Après avoir généré la première vidéo (HOME), utiliser "Save as Template"
   - Réutiliser ce template pour toutes les autres vidéos avec des ajustements mineurs

2. **Références de Style** :
   - Créer une image de référence avec les couleurs principales (turquoise, bleu, doré)
   - Télécharger cette référence pour chaque vidéo pour maintenir la cohérence

3. **Progression de Profondeur** :
   - Respecter l'ordre de profondeur dans les prompts
   - Cela créera une progression visuelle naturelle entre les pages

4. **Mouvements de Caméra** :
   - Varier les mouvements mais garder une vitesse lente et fluide
   - Éviter les mouvements brusques qui casseraient l'ambiance

## Notes Importantes

- **Coût** : Chaque vidéo coûte environ 180 crédits selon l'interface
- **Qualité** : Les vidéos générées seront en haute qualité, adaptées pour backgrounds web
- **Format** : Les vidéos seront au format MP4, prêtes à être intégrées
- **Loop** : S'assurer que chaque prompt mentionne "seamless loop" pour une boucle parfaite
- **Cohérence** : Utiliser le même style de référence pour toutes les vidéos pour une expérience fluide
